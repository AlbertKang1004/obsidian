## Regex

| Regular Expression                  | Strings that conform    |
| ----------------------------------- | ----------------------- |
| `[abc]`                             | a, b, c                 |
| `[abc]*`                            | "", aa, b, ab, abcacbaa |
| `^[a-z][a-zA-Z0-9]*$`               | csc207, albertKang11    |
| `[A-Z][0-9]+[A-Z]?`                 | A1,  A, Z9              |
| `[A-Z][a-z]*[0-9]+`                 | Hello123                |
| `([a-z]N[0-9]){3}`                  | bN3aN1kN9               |
| `(\d\d)([A-Z]([a-z]))\1\3(\d){3}\2` | 23Am23m987Am            |
### Quantifiers
>Specifies **how many times** the preceding element in a regex must occur for the pattern to match
- `*` **Zero** or **more** occurrence
- `?` **Zero** or **one** occurrence
- `+` **One** or **more** occurrence
- `{x}` exactly **x** occurrence
- `{x,}` **x** or more occurrences
- `{x, y}` between **x** and **y** occurrences
### Character Class
- `[a-z]` Any **lowercase** character
- `[A-Z]` Any **uppercase** character
- `[0-9]` Any **digit** character

- `.` Matches **everything**
- `\d` Matches **digits**
- `\D` Matches **non-digits**
- `\w` Matches **word characters** (alphabet + digit)
- `\W` Matches **non-word characters**
- `\s` Matches **any whitespace character**
- `\S` Matches **non-whitespace character**

### Anchor
- `^` At the **start** of the string
	- Note it has another usage: `[^abc]` Matches everything **except** a, b, and c
- `$` At the **end** of the string
### Grouping
- `\1` repeat **grouping number 1**, it has to match the previous substring
	- grouping count from opening braces `(`
### Logical Operators
- `|` Means **OR**
- `&&` Means **AND**
## Regex in Java
> There are 3 ways to use `regex` in Java.
### Use the `Pattern` and `Matcher` classes as follows
   ```java
import java.util.regex.*;

public class PatternMatcherExample {
    public static void main(String[] args) {
        Pattern p = Pattern.compile("a*b"); // a and ends with b
        Matcher m = p.matcher("aaaab");     // input string
        boolean matches = m.matches();      // full match required

        System.out.println(matches); // true
    }
}
```
### Use `Matcher.find` and `Matcher.group`
```java
import java.util.regex.*;

public class FindGroupExample {
    public static void main(String[] args) {
        Pattern p = Pattern.compile("\\d+");   // one or more digits
        Matcher m = p.matcher("Order 123, Code 456");

        while (m.find()) {                    // find each occurrence
            System.out.println(m.group());    // prints matched substring
        }
    }
}
```
### Use the `matcher` method from the `String` class 
```java
import java.util.regex.*;

public class StringMatcherExample {
    public static void main(String[] args) {
        String text = "abc123xyz";
        Matcher m = Pattern.compile("\\d+").matcher(text);

        if (m.find()) {
            System.out.println("Found number: " + m.group()); // prints 123
        }
    }
}
```

> In Java, instead of `\` we have to use `\\` **(very important)**
